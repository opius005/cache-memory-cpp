# cache-memory-cpp

### How do I get set up? ###

* Give hexadecimal format input in Hex_input.txt.
* Run Hex_to_bin.cpp file.
* Output will be Cache_input.txt file.
* Run Cache.cpp file.
* Output will be displayed in the terminal.
